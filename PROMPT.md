> 2. `01_AI_RULES.md` (Business Logic & Coding Standards)
> 3. `02_PROJECT_CHECKLIST.md` (Current Progress)
> 
> 
> Confirm that you understand this is an **RPA (Slaughterhouse) ERP** project and you will strictly follow the **Filament Shield RBAC** syntax.
> After confirming, please look at `02_PROJECT_CHECKLIST.md`, identify the first unchecked item in **Phase 1**, and guide me on how to implement it. Don't generate code yet, just explain the plan first."

###3. Prompt Jika Terjadi Bug / AI Melanggar Aturan (Correction)*Gunakan ini jika AI memberi kode yang salah (misal: pakai float, lupa permission, atau logic bisnis salah).*

**Kasus A: AI Lupa Syntax Permission (Laravel Shield)**

> **Prompt:**
> "⚠️ **STOP. You are violating the rules.**
> Check `01_AI_RULES.md` section 'RBAC & Visibility Implementation'.
> You implemented the `buying_price` column WITHOUT the `visible()` check. This is sensitive data.
> Refactor the code immediately to use: `->visible(fn () => auth()->user()->can('view_buying_price'))`.
> Do not ask for confirmation, just fix the code."

**Kasus B: AI Salah Tipe Data (Float vs Decimal)**

> **Prompt:**
> "⚠️ **Violation Detected.**
> You used `$table->float('weight')` in the migration.
> Refer to `00_SETUP_GUIDE.md`. We strictly use `decimal` for all weights and currency to avoid floating-point errors.
> Change it to `$table->decimal('weight', 10, 3)` (for precision). Correct the migration file now."

**Kasus C: Logic Produksi Salah (Assembly vs Disassembly)**

> **Prompt:**
> "Hold on. The logic you proposed seems to be 'Assembly' (combining items).
> Remember the Business Logic in `01_AI_RULES.md`: This is a **Poultry Slaughterhouse (Disassembly)**.
> One input (Live Chicken) becomes MANY outputs (Carcass, Feet, Head).
> Please rewrite the form schema to use a `Repeater` for the outputs, while keeping a single section for the Input Source."

###Tips Tambahan untuk Kamu:1. **Selalu Referensikan File:** Jika AI mulai "ngelantur", cukup ketik: *"Read 01_AI_RULES.md again."*
2. **Commit Message:** Kamu juga bisa minta AI buatkan commit message. *"Generate a git commit message for these changes."*
3. **Dokumentasi:** Setelah fitur besar selesai, minta AI mencatatnya. *"Please summarize what we did today and update the checklist."*
